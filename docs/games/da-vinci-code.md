# 达芬奇密码

推理对手隐藏的数字牌，率先把对手全部「逼出局」的人获胜。

## 牌组

- 黑色 0–11、白色 0–11，共 24 张数字牌。
- 每张牌的「背面颜色」是公开的，但**数字对其他玩家隐藏**。
- 同一玩家的牌始终按升序排列：先比数字，数字相同时**黑色在前、白色在后**。

## 发牌

- 2 人：每人 4 张；3–4 人：每人 3 张。
- 其余牌放入牌堆。

## 回合流程

1. **抽牌**：轮到的玩家从牌堆摸一张牌（牌堆空则跳过），只有自己能看到数字。
2. **猜测**：指定一名对手的某张暗牌，说出你认为的数字（颜色已公开）。
   - **猜中**：该暗牌翻开（所有人可见）。你可以选择**继续猜测**，或**停止结算**。
   - **猜错**：把本回合摸到的牌**正面朝上**插入自己的牌列（数字暴露），回合结束。
3. **结算**：停止猜测时，把摸到的牌**正面朝下**插入自己牌列的正确位置，轮到下一位。
4. 当一名玩家的所有牌都被翻开时即出局；最后剩下的玩家获胜。

> 牌堆为空时若猜错，则改为亮出自己最左侧的一张暗牌作为惩罚。

## 隐藏信息（防作弊）

服务器对每个客户端**单独下发**经过脱敏的游戏状态：对手的暗牌只会下发颜色（数字为 `-1`，前端显示 `?`），牌堆与他人摸到的牌的数字也不会发送到前端。

## 推理提示

轮到你猜测并选中一张暗牌后，界面会高亮「根据已知信息仍然可能的数字」。该推理基于：

- 牌的已知颜色；
- 每个（颜色, 数字）组合全局唯一（已出现的数字可被排除）；
- 牌列严格升序所带来的位置上下界约束。

当某张暗牌只剩 1 个可能值时即可必中。

## 电脑 AI 行为

| 难度 | 行为 |
|------|------|
| 简单 | 经常随机乱猜，几乎只在必中时才继续 |
| 普通 | 多数情况下使用推理，必中才继续 |
| 困难 | 稳定使用推理，候选≤2 时也会赌一把继续 |
| 专家 | 几乎不失误，候选≤3 时也会继续施压 |

## 人数要求

- 最少 2 人，最多 4 人。
